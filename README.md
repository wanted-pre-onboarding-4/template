# Typescript Template

## installed dependencies
- react-router-dom
- axios
- styled-components
- eslint
- prettier
- husky

## Usage
1. `Clone` the repository
   ```markdown
   $ git clone https://github.com/wanted-pre-onboarding-4/template.git
   ```
2. `Set` remote add origin

3. `Install` dependencies
   ```markdown
   $ npm install
   ```
4. `Push` to your project github
